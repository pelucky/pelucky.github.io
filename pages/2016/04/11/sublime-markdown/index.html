Sublime Markdown

Date Mon 11 April 2016 Tags sublime / markdown

之前的Markdown编辑器都是使用马克飞象，今天折腾了一下sublime，查到可以修改为Markdown编辑器，记录下...

0x01 Package Control

推荐使用官方的代码安装，在sublime里面按ctrl+`打开控制台，在控制台里面输入如下安装Package Control。

import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

0x02 Markdown Editing

  1. 安装Markdown Editing，在sublime里面按ctrl+shift+p;
  2. 输入Package Control: Install Package;
  3. 输入Markdown Editing安装它，安装完重启sublime；

0x03 Markdown Preview

  1. 安装Markdown Preview,同上，选择Markdown Preview，重启sublime；

0x04 Usage

  1. 设置Markdown Editing，使用ctrl+shift+p,设置Set Syntax: Markdown
  2. 设置Markdown Preview, 使用ctrl+shift+p, 选择Preview in browser,选择以Github或者Markdown的格式显现出来；
  3. 弹出网页浏览；
