安装Django小记

Date Mon 14 September 2015 Tags Python / setuptools

主要是局域网蛋疼，无pip，yum中不包含Django，故只能手动安装

  1. 首先查看Python版本，CentOS6.5自带Python2.6，没戏，需要重新安装2.7以上版本；
  2. 下载Python2.7.10，源码安装，三部曲走起...
  3. 安装Django，提示需要安装setuptools，去下载，从头来；
  4. 安装setuptools，外网只需要一个指令，即可自动安装，内网蛋疼，/usr/local/python-2.7.10/bin/python setup.py install，提示如下：

简而言之就是没有zlib，网上查找再三，找到解决方法：

yum install zlib
yum install zlib-devel

再重新编译python2.7，再三部曲 1. 安装setuptools，成功； 2. 安装django，成功； 3. 如此蛋疼的问题，谁会想得到....
